# vodka
Конвертер квестов из `Текстовые квесты - играй и пиши` в `Meadner`

## Инструкция по использованию
* Вам необходим любой json файл квеста из ТК. Его можжно получить из assets внутри приложения либо вытащить прямо из серверов.
* Желательно переименовать его так, чтобы формат был .txt
* Откройте приложение, введите название квеста в Meadner, нажмите зелёную кнопку и загрузите .txt файл
* Конвертер выполнит конвертацию и выдаст вам .zip архив
* Переименуйте его в .mnd и импортируйте в Meadner

## Предупреждение
> Разработчик КРАЙНЕ осуждает воровство чужого авторского труда. Используйте конвертер только для своих квестов либо не выкладывайте в маркет работы других авторов. Спасибо.

## Возможности на данный момент (Будет обновляться)
* Базовый функционал перехода между страницами
* Разделение квеста на главы по 100 страниц (Нод)

## Сборка
Рекомендуется использовать **gradle** для сборки проекта.
